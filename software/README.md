# Workshop Software requirements
The workshop will require us to have some software. Here you will find some instructions to get the software and set it up to be used in the workshop.

**Please make sure to come to the workshop day with at least the following software installed or downloaded as indicated.**

Minimum required software
- [Grafana](./grafana/README.md) (preferable to use binaries)
- [jmeter](./jmeter/README.md)
- [InfluxDB OSS 1.x](./influxdb/README.md)(https://www.influxdata.com/downloads/) (**PLEASE DOWNLOAD v1.X binaries** Do not download v2.x nor v3.x)
- [k6](./k6/README.md) (It will be installed during the workshop)

Prefered but not mandatory
- NodeJs
- Prometheus
