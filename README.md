# Perf Workshop
This is the repository for the Modern Performance Workshop

This repo is divided in some sections that will be used for the workshop.

- [Software needed](./software/README.md)
- [Examples](./examples/README.MD)
- [WebService](./webservice/README.md)

## Software needed
The workshop will require us to have some software. In [this](./software/README.md) directory you will find some instructions to get the software and set it up to be used in the workshop.

**Please make sure to come to the workshop day with at least the following software installed or downloaded as indicated.**

Minimum required software
- [Grafana](./software/grafana/README.md) (preferable to use binaries)
- [jmeter](./software/jmeter/README.md)
- [InfluxDB OSS 1.x](./software/influxdb/README.md)(https://www.influxdata.com/downloads/) (**PLEASE DOWNLOAD v1.X binaries** Do not download v2.x nor v3.x)
- [k6](./software/k6/README.md) (It will be installed during the workshop)

Prefered but not mandatory
- NodeJs
- Prometheus
